# student-performance-analysis.
This project analyzes the relationship between study hours and student marks using a real dataset.

# Objective
To understand how study time affects academic performance.

# Tools Used
- Python
- Pandas
- Matplotlib

# Analysis Performed
- Scatter plot (Hours vs Marks)
- Bar graph (Average marks by study hours)
- Correlation analysis
- Trend line visualization

# Key Insights
- Positive correlation between study hours and marks
- Students who studied more scored higher on average
- Consistent study improves performance

# Conclusion
Study hours have a significant impact on student performance.

# Files
- `project.ipynb` → Main analysis
- `dataset.csv` → Dataset used
